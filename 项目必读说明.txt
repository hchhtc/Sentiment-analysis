2. 基于PyQt5搭建的4分类情感分析系统演示
环境依赖配置：
本地需要先下载paddle和paddle'hub以及Pyqt5等环境依赖包。

环境配置指导：
建议先使用conda create创建一个环境再在该环境上完成环境依赖包的配置
// 首先设置清华源镜像加速
 conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
 conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
 conda config --set show_channel_urls yes
//conda创建一个名为test，python版本为3.6的虚拟环境
conda create -n test python=3.6
//进入刚才创建的test环境,在里面安装所需依赖包：  
activate test

paddle下载请移步至https://www.paddlepaddle.org.cn/install/quick?docurl=/documentation/docs/zh/install/pip/windows-pip.html，根据官方提供的下载教程完成paddle2.0.2下载

其他python依赖包下载指导:
paddlehub下载：
pip install -U paddlehub -i https://pypi.tuna.tsinghua.edu.cn/simple
Pyqt5下载：
pip install PyQt5 -i  http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
其他依赖包下载：
pip install csv -i  http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
pip install xlrd -i  http://pypi.douban.com/simple/ --trusted-host pypi.douban.com

Note:
在完成环境配置及将训练好的模型参数放入ckpt文件夹下后运行gui.py即可得到4分类情感分析系统可视化界面。

4分类情感分析系统功能介绍：
a.单条文本情感分类：
功能介绍：对用户输入的单条文本进行情感分类预测。
使用说明：在输入文本框内输入要进行情感分类的文本数据后点击单条文本情感分类按钮，运行完成后显示预测的情感类别标签。
b.批量文本情感分类：
功能介绍：对用户导入的文本excel文件进行批量预测并输出预测结果文件。
使用说明：
首先用户需要点击选择Excel文件按钮选择要进行批量分类的Excel文本文件。接着设置好预测结果文件输出路径。完成后点击进行批量情感分类按钮，系统将读取用户选择的Excel文件数据并逐行预测输出预测结果文件。
